How Cities Form Endogenously: The Complete Mechanism

1
Initial State: Homogeneous Equilibrium
Start with a perfectly uniform distribution y(x,0) = y₀ + ε where ε is tiny random noise. No cities exist yet. Classical models (Krugman, Henderson) require ex ante heterogeneity or productivity shocks. We require nothing.
2
Linear Stability Analysis: Turing Instability
Linearize around y₀: perturbation δy = e^(σt)e^(ik·x). Growth rate: σ(k) = M(-κ|k|² + λK̂(k)).
  • If λ < λ_c: σ(k) < 0 for all k → stable, no pattern
  • If λ > λ_c: σ(k) > 0 for some k → unstable, cities emerge!
The critical value is λ_c = mink κ|k|²/K̂(k) where K̂(k) > 0.
3
Mexican Hat Kernel: Short Repulsion + Long Attraction
K(x) = exp(-|x|²/2σ₁²) - 0.5·exp(-|x|²/2σ₂²) with σ₂ > σ₁.
  • Short range (|x| < σ₁): Positive → attracts nearby population
  • Medium range (σ₁ < |x| < σ₂): Negative → repels moderate distance
  • Long range (|x| > σ₂): Near zero → no interaction
This creates self-organized criticality: cities cluster but maintain separation.
4
Wavelength Selection: Characteristic City Scale
The mode k_max = argmaxk σ(k) grows fastest. City spacing: ℓ = 2π/k_max. This is endogenous scale selection—the characteristic city size emerges from parameter values (λ, κ, σ₁, σ₂), not from external geography or productivity.
5
Nonlinear Saturation: Congestion Limits Growth
The cubic term -α(y-y₀)³ prevents unbounded growth. As cities grow (y increases), congestion forces balance topological attraction. System reaches a stationary pattern y* with:
0 = κΔy* + λ(K*y*) - α(y*-y₀)³
This is a stable nonlinear equilibrium with multiple cities at characteristic scale ℓ.
6
Solving Starrett's Impossibility Theorem
Starrett (1978) proved cities cannot form in models with:
  • Homogeneous space
  • Convex preferences
  • Local interactions only
We escape via nonlocal topological coupling λ(K*y). The convolution (K*y) creates long-range forces that break the impossibility. No agglomeration externalities, increasing returns, or transport costs needed.
